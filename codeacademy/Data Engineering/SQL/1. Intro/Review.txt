Relational database: database that organize information into one or more tables.
Talbe: collection of data organized into rows(records) and columns

CREATE TABLE
 creates a new table.

INSERT INTO
 adds a new row to a table.

SELECT
 queries data from a table.

ALTER TABLE
 changes an existing table.

UPDATE
 edits a row in a table.

DELETE FROM
 deletes rows from a table.
